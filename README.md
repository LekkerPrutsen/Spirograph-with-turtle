## Usage
```
Usage: python[3] spirograph_turtles.py [-a] <drawing>
	-a      : animate the drawing (default no animation)
	drawing : one of
		blue_shades
		donut
		hurricane
		purple_pentagons
		silver_angles
		square_shell
		sunflower
		windmill
```
